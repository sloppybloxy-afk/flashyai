# Flashly - Study Smarter with Spaced Repetition

A beautiful flashcard application built with HTML, CSS, and JavaScript.

## Features
- 📚 Create study decks
- ✨ Generate flashcards with AI
- 📊 Track study statistics
- 💾 Local data persistence
- 📱 Fully responsive design

## Live Demo
Visit: https://flashyai.github.io/

## Getting Started
1. Clone this repository
2. Open `index.html` in your browser
3. Start creating decks and studying!

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
